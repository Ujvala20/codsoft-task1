# codsoft-task1
Titanic Survival Prediction

This project predicts whether a passenger survived the Titanic disaster using machine learning. It includes data cleaning, exploratory analysis, feature engineering, and a classification model trained on the Kaggle Titanic dataset.

Features

Data preprocessing and handling missing values

Visual analysis of survival patterns

Machine learning model for survival prediction

Evaluation of model performance

Technologies

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

How to Run

Open the notebook in Google Colab and upload the Titanic dataset when prompted
